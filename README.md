# fastApiTutorial
- fastapi 를 사용한 간단한 api 만들기

### 1.  run
- <code> source ./venv/bin/activate </code>
- <code> python3 -m uvicorn main:app --reload </code>

### 2. host
- <code> http://127.0.0.1:8000 </code>

### 3.  Swagger UI
- http://127.0.0.1:8000/docs
